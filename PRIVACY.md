# Política de Privacidade - Music Player

**Última atualização:** Fevereiro 2025

## Resumo

Este aplicativo foi desenvolvido para uso pessoal e **NÃO coleta, armazena ou transmite nenhum dado pessoal**.

## Dados Coletados

**Nenhum.** Este aplicativo:

- ❌ NÃO coleta informações pessoais
- ❌ NÃO rastreia sua localização
- ❌ NÃO acessa sua lista de contatos
- ❌ NÃO envia dados para servidores externos
- ❌ NÃO possui analytics ou telemetria
- ❌ NÃO requer login ou cadastro
- ❌ NÃO possui anúncios

## Funcionamento 100% Offline

Este aplicativo funciona **completamente offline**:

- Todas as músicas são lidas diretamente do seu dispositivo
- Playlists são salvas apenas localmente no seu dispositivo
- Nenhuma conexão com internet é necessária ou utilizada

## Permissões Solicitadas

O aplicativo solicita apenas as permissões mínimas necessárias:

### Android
- **READ_EXTERNAL_STORAGE** / **READ_MEDIA_AUDIO**: Para acessar seus arquivos de música
- **FOREGROUND_SERVICE**: Para reprodução em segundo plano
- **WAKE_LOCK**: Para manter a reprodução ativa com a tela desligada

### iOS
- **NSAppleMusicUsageDescription**: Para acessar sua biblioteca de mídia

## Armazenamento Local

Os únicos dados armazenados são:
- Suas playlists (salvas no armazenamento local do app)
- Preferências de reprodução (volume, shuffle, repeat)

Estes dados nunca saem do seu dispositivo.

## Código Aberto

Este aplicativo não possui código malicioso ou oculto. Todo o funcionamento é transparente e focado exclusivamente em reproduzir suas músicas locais.

## Contato

Este é um aplicativo para uso pessoal. Não há suporte comercial ou coleta de feedback.
